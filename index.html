Mind Hacks for Web Developer

DevMind suggests this four mind hacks for web developer for maximum productivity.

Mental Models

For example: Know how a browser process HTML, CSS and Javascript.

RD Debugging

Rubber Duck Debugging is: Explaining your codes, problems or logics out loud.

Incremental Learning

"Learn Just Enough". It means: Learn, not everything, but what you need to complete task.

Code Review

Approach code reviews with a mindsets of growth. Treat every opportunity as a chance of learning.
